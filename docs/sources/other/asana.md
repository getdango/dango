# Asana

**Category:** Business & CRM | **Auth:** API Key | **Wizard:** Disabled

## Status

Asana is registered but **disabled** in the wizard. The Asana SDK dependency was removed from the vendored dlt source, so the connector no longer functions.

## Workaround

Use the [REST API source](../non-wizard-sources.md#rest-api) to connect to Asana's REST API directly, or use the [dlt Native source](../non-wizard-sources.md#dlt-native-source) for a custom integration.
