# Jira

**Category:** Business & CRM | **Auth:** API Key | **Wizard:** Disabled

## Status

Jira is registered but **disabled** in the wizard. The vendored dlt source uses an incorrect endpoint, causing sync failures.

## Workaround

Use the [REST API source](../non-wizard-sources.md#rest-api) to connect to Jira's REST API v3 directly, or use the [dlt Native source](../non-wizard-sources.md#dlt-native-source) for a custom integration.
